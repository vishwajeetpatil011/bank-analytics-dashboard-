# 📊 Bank Analytics Dashboard

![Bank Analytics Dashboard](Images/Screenshot%202026-07-15%20001732.png)

## 📌 Project Overview

The **Bank Analytics Dashboard** is an end-to-end Business Intelligence project designed to analyze banking data and generate meaningful insights using **Excel, SQL, Power BI, and Tableau**.

The project focuses on understanding customer transactions, credit/debit patterns, loan performance, repayment behavior, and key financial KPIs through interactive dashboards.

The goal is to transform raw banking data into actionable business insights that can support data-driven decision-making.

---

# 🎯 Business Objectives

- Analyze banking transaction performance.
- Understand customer financial behavior.
- Track loan growth and repayment trends.
- Identify important financial KPIs.
- Build interactive dashboards for business users.
- Generate insights to improve banking operations.
